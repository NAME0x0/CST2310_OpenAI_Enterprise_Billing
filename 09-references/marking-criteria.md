## Quick Self-Assessment Checklist

Use this before submission to ensure all marking criteria are addressed:

### Front Matter

- [ ] Front cover sheet with group members, module, lab day/time?
- [ ] Abstract?
- [ ] Table of Contents?
- [ ] Table of Figures?

### Introduction (part of Case Study, 10 marks)

- [ ] Business Information System (BIS) defined with reference?
- [ ] Transaction Processing System (TPS) defined with reference?
- [ ] Management Information System (MIS) defined with reference?
- [ ] Operational documents discussed?
- [ ] Databases and management reports mentioned?
- [ ] Aims and objectives stated?
- [ ] Submission structure (chapter breakdown) described?

### Project Management (20 marks)

- [ ] Project management defined with reference?
- [ ] Activity List with task descriptions, start/end dates, predecessors, owners?
- [ ] Work Breakdown Structure with 2-3 levels hierarchy?
- [ ] Gantt chart with critical path and slack time shown?
- [ ] Critical activities identified?
- [ ] Risk matrix with likelihood, impact, descriptions, mitigation?

### Case Study (10 marks)

- [ ] Half to two-thirds page narrative description?
- [ ] Business case statement (half A4)?
- [ ] Functional requirements defined and listed?
- [ ] Non-functional requirements defined and listed?
- [ ] Data collection methods discussed with references?

### UML Schematic Models (45 marks total)

- [ ] UML defined with reference?
- [ ] Each schematic model type defined?
- [ ] 4-6 Use Case Diagrams with actors, system boundary, include/extend?
- [ ] 4-6 Use Case Descriptors with clear steps and correlation to diagrams?
- [ ] Class Diagram with attributes, operations, relationships, multiplicity?
- [ ] Corresponding Entity-Relationship Diagram?
- [ ] 4-6 Activity Diagrams with decision symbols, start/stop, swim lanes?
- [ ] 4-6 Sequence Diagrams with objects, messages, time ordering?
- [ ] 4-6 Collaboration Diagrams corresponding to sequence diagrams?
- [ ] 4-6 User Interfaces incorporating Nielsen's Heuristics?

### Law and Ethics (10 marks)

- [ ] GDPR narrative description (how it is implemented)?
- [ ] Security measures to safeguard personal data?
- [ ] Policies concerning Equality legislation?
- [ ] Quality references with in-text citations?

### Conclusions

- [ ] Links to other modules (Java, HCI, PM)?
- [ ] Evaluation of the group project experience?
- [ ] Conclusion summarising key findings?

### References

- [ ] Harvard referencing format throughout?
- [ ] All in-text citations appear in the reference list?

### Appendix

- [ ] Minutes of group meetings (weeks 7-12)?
- [ ] Self-assessment forms (one per member)?
- [ ] Reflective statements (A4 max, one per member)?
- [ ] Peer assessment forms (one per member)?
- [ ] Presentation slides?

### Presentation (5 marks)

- [ ] TOC and TOF present?
- [ ] All diagrams clearly labelled with figure numbers?
- [ ] Clear section delineation?
- [ ] Good standard of English?
